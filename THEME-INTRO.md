# Atrom - 现代 Astro 主题介绍

## 主题概述

Atrom 是一款基于 Astro v6 构建的现代化、响应式网站主题，融合了极简设计美学与强大功能，为企业和个人提供了理想的线上展示平台。

## 核心价值

### 🎯 精心设计的用户体验
- **极简美学**：采用现代简约设计风格，去除冗余元素，突出核心内容
- **流畅交互**：精心设计的过渡动画和微交互，提升用户浏览体验
- **专业排版**：使用 Inter 字体家族，确保跨平台的一致可读性

### 🌐 全球化支持
- **双语言架构**：内置中文和英文完整支持
- **无缝切换**：导航栏语言切换器，一键切换语言版本
- **本地化优化**：针对不同语言的排版和内容展示进行优化

### 🚀 性能卓越
- **Astro 驱动**：利用 Astro 的静态站点生成能力，实现极速加载
- **响应式设计**：在移动设备、平板和桌面端均有出色表现
- **优化加载**：智能资源加载策略，确保最佳性能

### 💡 功能完备
- **多页面架构**：首页、产品、价值和故事四大核心页面
- **现代化组件**：Hero 区域、特性展示、价值主张、客户故事等
- **品牌化设计**：支持自定义品牌标识和颜色方案

## 技术亮点

- **Astro v6 兼容**：基于最新版本 Astro 构建，享受最新特性
- **服务器端渲染**：支持 SSR 功能，提升动态内容性能
- **Cloudflare Pages 优化**：内置 Cloudflare 部署配置，一键部署
- **模块化结构**：清晰的项目组织，易于维护和扩展

## 适用场景

- **企业官网**：展示公司形象、产品和服务
- **产品展示**：突出产品特性和价值主张
- **品牌宣传**：建立专业的品牌在线形象
- **个人作品集**：展示个人项目和成就

## 快速开始

```bash
# 1. 克隆项目
git clone <repository-url>
cd atrom

# 2. 安装依赖
pnpm install

# 3. 启动开发服务器
pnpm dev

# 4. 构建生产版本
pnpm build
```
